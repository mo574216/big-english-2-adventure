# Big English 2 Reference Package

This repository package is a structured curriculum reference for building an original static web application provisionally titled **English Adventure 2**.

The package is based on the syllabus progression of *American Big English 2 Student's Book, Second Edition*. It contains curriculum facts, short target-language patterns, vocabulary, learning outcomes, page references, implementation constraints, and activity-design guidance.

It does **not** contain the textbook PDF, scanned pages, publisher illustrations, original audio, songs, complete stories, or substantial verbatim exercises.

## Files

- `curriculum/course-structure.json`: authoritative machine-readable curriculum.
- `curriculum/course-structure.schema.json`: lightweight validation schema.
- `docs/curriculum-map.md`: human-readable course map.
- `docs/content-guidelines.md`: copyright, privacy, pedagogy, and content rules.
- `docs/activity-design-guide.md`: activity engines and assessment behaviour.
- `docs/implementation-plan.md`: staged development plan.
- `TASKS.md`: implementation checklist.
- `CODEX-FIRST-TASK.md`: first Codex task prompt.
- `README-FA.md`: Persian upload and execution instructions.

## Authoritative source order

When instructions conflict, Codex should follow this order:

1. `docs/content-guidelines.md`
2. `curriculum/course-structure.json`
3. `docs/curriculum-map.md`
4. `docs/activity-design-guide.md`
5. `docs/implementation-plan.md`
6. `TASKS.md`

## Product positioning

The application must be an **unofficial, original practice companion**, not a digital copy of the book and not an official Pearson product.
